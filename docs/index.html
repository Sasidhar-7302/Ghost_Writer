One-command install

Deploy directly to your local machine. No complex setup, no proxy required.

Windows x64

powershell -NoProfile -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/Sasidhar-7302/Ghost_Writer/main/install.ps1 | iex"

Installs per-user, verifies the release checksum, and upgrades in place.

macOS arm64

curl -fsSL https://raw.githubusercontent.com/Sasidhar-7302/Ghost_Writer/main/install.sh | bash

Installs into ~/Applications, verifies the release checksum, and upgrades in place.

Local-first options

You can run local models and local speech pipelines where configured, or use cloud providers when you choose to.

How is data handled?

Credentials are stored with OS secure storage when available. Cloud providers only receive requests when you configure and use them.

What if install warnings appear?

v1.0.0 is unsigned. Use the published checksum to verify the download, then follow the troubleshooting guide for platform-specific launch prompts.

Where do I get support?

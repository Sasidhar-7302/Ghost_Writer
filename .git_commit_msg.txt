Ghost Writer v2.0.0 — AI Interview Copilot

Production-ready AI copilot for high-stakes interviews and meetings.

Features:
- GPU-accelerated local transcription via whisper-server (whisper.cpp)
- Persistent server mode: model stays in GPU VRAM for ~1-2s transcription
- Multi-provider LLM support (Claude, GPT, Gemini, Groq, DeepSeek, Ollama)
- RAG pipeline with local embeddings (all-MiniLM-L6-v2)
- Native Rust audio capture (WASAPI) with DSP pipeline
- Invisible overlay UI with screen-share protection
- Interview-specific AI modes (What to Answer, Shorten, Recap, Follow Up)
- Cost tracking, rate limiting, structured logging
- Setup wizard for first-run onboarding
- CI/CD with GitHub Actions

Tech stack: Electron, React, TypeScript, Rust, whisper.cpp, SQLite
